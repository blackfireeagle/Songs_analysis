# Анализ текстов песен Billie Eilish и Adele

## Автор проекта

- Юсупова Диана (468162)

## Цель проекта
Сравнительный анализ текстов песен двух современных исполнительниц - Billie Eilish и Adele - с выявлением лексических особенностей, тематических предпочтений и стилистических различий. Проект направлен на определение ключевых характеристик текстового контента каждой исполнительницы и выявление факторов, влияющих на их уникальный стиль.

## Обзор исследования
Проект представляет собой комплексный анализ текстового содержания песен двух популярных исполнительниц современной музыки. Основное внимание уделяется:

- Лексическому разнообразию (анализ TTR - Type-Token Ratio)
- Частоте употребления слов (облака слов, топ-20 слов)
- Длине текстов (количество слов и символов)
- Сравнению альбомных и сингловых релизов

## Ключевые этапы исследования

### 1. Сбор данных
Данные были собраны с использованием Genius API через библиотеку lyricsgenius. Собраны тексты всех основных альбомов и синглов обоих исполнительниц.

Использованные технологии:
- Библиотека lyricsgenius для доступа к API Genius
- Кастомные обработчики для коллабораций и особых случаев
- Задержки между запросами для соблюдения лимитов API

### 2. Очистка и подготовка данных
Тексты песен были очищены от:
- Технической информации (например, "Read More")
- Текста в квадратных скобках ([Verse 1] и т.д.)
- Лишних пробелов и пустых строк

Результаты сохранены в структурированные JSON-файлы с разделением по альбомам.

### 3. Анализ и визуализация
В ходе анализа были выявлены следующие ключевые закономерности:

#### Лексические особенности:
- Billie Eilish демонстрирует более высокий TTR (0.563 против 0.494 у Adele), что указывает на большее лексическое разнообразие
- В текстах Adele чаще встречаются слова, связанные с эмоциями (heart, cry) и устойчивыми выражениями
- Billie Eilish использует больше разговорных выражений и слов, передающих неуверенность (maybe, guess)

#### Структурные особенности:
- Песни Adele в среднем длиннее (318 слов против 243 у Billie Eilish)
- Распределение длины песен у Adele более однородное
- Billie Eilish показывает больший разброс - от очень коротких до длинных композиций

### 4. Интерактивный дашборд
Разработан интерактивный дашборд, включающий:
- Сравнительную статистику по количеству слов и символов
- Облака слов для каждого исполнителя
- Топ-20 наиболее употребляемых слов
- Сравнение лексического разнообразия (TTR)
- Анализ удобочитаемости текстов

## Ключевые выводы

1. **Лексическое разнообразие**: Billie Eilish использует более разнообразную лексику, что соответствует её экспериментальному стилю, в то время как Adele придерживается более традиционного подхода с повторением ключевых эмоциональных слов.

2. **Длина текстов**: Adele пишет более длинные и содержательные тексты, Billie Eilish демонстрирует больший разброс - от минималистичных до развернутых композиций.

3. **Стилистические различия**: 
   - Billie Eilish: интимный, неформальный стиль с элементами разговорной речи
   - Adele: мощный, драматический стиль с акцентом на эмоциональную выразительность

4. **Тематические предпочтения**:
   - Обе исполнительницы исследуют темы любви и отношений
   - Billie Eilish чаще обращается к темам неуверенности и интроспекции
   - Adele фокусируется на сильных эмоциях и драматических переживаниях

## Направления дальнейшего развития

1. Расширение анализа на большее количество исполнителей для выявления жанровых закономерностей
2. Анализ эволюции стиля исполнителей по хронологии альбомов
3. Исследование корреляции между характеристиками текстов и коммерческим успехом песен
